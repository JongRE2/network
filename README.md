### HTTP 상캐코드

+ HTTP상태코드 소개
+ 2xx - 성공
+ 3xx - 리다이렉션1 - 영구 리다이렉션
+ 3xx - 리다이렉션2 - 일시 리다이렉션
+ 4xx - 클라이언트 오류, 5xx - 서버 오류
